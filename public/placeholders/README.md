## Recommended Specifications

### Videos
- Format: MP4 (H.264) or WebM
- Resolution: 1920x1080 for hero videos, 1280x720 for gallery
- Duration: 10-30 seconds for loops
- File size: Keep under 10MB for quick loading

### Images
- Format: JPG for photos, PNG for graphics with transparency
- Resolution: 1920x1080 for hero images, 800x600 for thumbnails
- Optimize for web (use tools like TinyPNG)

### Thumbnails
- Aspect ratio: 4:3 or 16:9
- Resolution: 400x300 minimum
- File size: Under 100KB each
